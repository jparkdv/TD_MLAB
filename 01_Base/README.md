# 🧪 TD_MLAB: Path to a Global Technical Artist
**Articulating Art through Tech: The Foundation for a Technical Artist.**

`TD_MLAB` is a creative laboratory dedicated to exploring the intersection between rigorous engineering and expressive media art. This archive serves as the engine room for mastering TouchDesigner, AI-driven generative art, and responsive interactive systems.

---

## 📂 Laboratory Structure

### 🏛️ 01_Base: Logic & Language
**"Every complex interaction is built on simple logical blocks."**
- **Purpose:** This is a dedicated space for hands-on practice and deep-diving into the core mechanics of various programming languages.
- **Content:** It documents a series of **mini-projects** built from scratch, translating fundamental syntax into functional logic.
- **Goal:** To ensure a rock-solid technical foundation capable of supporting international-scale gallery installations and complex digital ecosystems.

### 🌊 02_Vibe_Lab: Creative Flow
**"Where Logic meets Emotion."**
- **Purpose:** A creative playground where the technical skills honed in '01_Base' are transformed into intuitive and sensory **Vibe Coding** projects.
- **Content:** This section archives experimental media artworks, including p5.js interactions, generative patterns, and responsive web aesthetics.
